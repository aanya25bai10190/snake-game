# 🐍 Neon Serpent

A modern reimagining of the classic Snake game built with vanilla JavaScript, featuring smooth animations, particle effects, power-ups, dynamic difficulty scaling, and a polished UI experience.

## 🎮 Play Now

**Live Demo:**
https://aanya25bai10190.github.io/snake-game/snake-game/

---

## ✨ Features

### Core Gameplay

* Classic Snake mechanics
* Increasing difficulty as score rises
* High score tracking using Local Storage
* Keyboard controls (Arrow Keys / WASD)

### Visual Effects

* Particle explosions when food is collected
* Smooth animations and transitions
* Dynamic glowing game elements
* Responsive UI interactions
* Animated overlays

### Power-Ups

* 🟣 Slow Motion Mode

  * Temporarily slows game speed
  * Helps navigate difficult situations

* 🔵 Ghost Mode

  * Allows passing through walls temporarily
  * Strategic escape mechanic

### Modern UI

* Glassmorphism-inspired interface
* Animated score indicators
* Neon visual theme
* Responsive layout
* Smooth hover interactions

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* Canvas API
* Local Storage API

No frameworks or external libraries were used.

## 🚀 Local Setup

Clone the repository:

```bash
git clone https://github.com/aanya25bai10190/snake-game.git
```

Navigate to the project folder:

```bash
cd snake-game
```

Run with Live Server or any local development server.
